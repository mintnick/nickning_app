EVE 设置管理工具

简介

大部分 EVE 玩家都有多个账号和角色，这个工具可以让玩家一键同步所有账号和角色的设置，包括界面布局、总览、快捷键等。

本工具不违反任何 CCP Games 的规定，其实现原理是：

  • 每一个登录过的角色和账号，在本地会产生配置文件，下次登录时读取对应的游戏内设置
  • 通过复制、粘贴、重命名配置文件的方式，玩家可以让多个角色使用一样的设置文件，省去了手动调整每个角色的麻烦
  • 这个工具把读取文件、复制粘贴等操作进行了自动化，达到一键设置的效果

功能类似的工具在多年前就已经存在，只不过使用起来不太方便，功能也较简陋，所以我重做了一个。

使用

  • 左上角选择服务器，右上角选择语言
  • 中间路径菜单会列出默认位置和名称的设置文件夹，如果你曾经在不同盘符安装过多个客户端，设置文件夹的位置可能会有所不同，你可以通过下面的“选择文件夹”按钮手动指定设置文件夹
  • “打开文件夹”会在资源管理器中打开对应文件夹，“备份”会在设置文件夹里创建一个子文件夹，并把设置文件复制进去，“清空缓存”用于清空角色列表里的角色名、备注等信息
  • 在下面的角色/账号中，选择一个作为模版，然后在最下方“覆盖所有”或者“覆盖...”选择哪些对象会被覆盖
  • 备注可以用来标注账号对应的邮箱、功能，因为账号的数字 ID 是无法通过 ESI 获取信息的
